## What is the Demo Connection?

The Demo connection is a built-in testing environment that comes pre-loaded with realistic sample affiliate data. It lets you explore AffZero's full feature set — building automations, testing variables, sending emails — without needing access to a real tracking platform.

**The Demo connection does not count toward your connection limit.**

---

## What Data Does It Include?

The Demo connection returns a realistic dataset including:

- A set of sample affiliates with names, clicks, conversions, revenue, and payouts
- Offer-level data
- Date-range-based breakdowns

The data is static (it doesn't change day to day), but it's structured exactly like real tracker data, so anything you build with it will work the same way with a live connection.

## Available Stat Types

| Stat Type | What it shows |
|---|---|
| **By Affiliate** | Sample affiliate performance — revenue, payout, clicks, conversions per affiliate |

More stat types may be added to the Demo connection as new platform integrations are built.

---

## How to Add the Demo Connection

1. Go to **Connections** in the left sidebar
2. Click **Add Connection**
3. Select **Demo** from the platform list
4. Enter any name you like (e.g. `Demo`)
5. Click **Save**

No API key or base URL is required.
