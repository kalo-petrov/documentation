## Coming Soon

AffZero's Everflow integration is currently in development. If you use Everflow and want to be notified when the integration launches, please reach out via the in-app chat or contact us directly.

## What Will Be Supported

When the Everflow integration launches, AffZero will support:

- Pulling stats **By Affiliate**
- Pulling stats **By Campaign**
- Pulling stats **By Date**

The connection will use your **Everflow API key**, which can be found in your Everflow portal under Settings → API.

## In the Meantime

If you need affiliate stats automation now, consider:

- Using the [Demo connection](/connections/demo) to explore AffZero's features with sample data
- Contacting us to be added to the Everflow beta waitlist
